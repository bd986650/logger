# Logger - тестовое задание
Скрипт для обработки лог-файлов и подсчёта статистики

Программа читает файл построчно, парсит логи и выводит:
- общее количество строк  
- количество строк с уровнем `ERROR`  
- топ-3 пользователей с наибольшим числом ошибок  

### Что реализовано:
- Скрипт
- Тесты
- ESLint + Prettier
- Docker

### Основные файлы
- readLines — потоковое чтение файла  
- parseLine — парсинг строки  
- stats — агрегация статистики  
- formatOutput — форматирование результата  
- runScript — связывает всю логику  
- index — точка входа  

---

## Установка и запуск
```bash
npm install
```

```bash
npm run dev -- ./example_data/data.txt
```

## Тесты
```bash
npm test
```
- unit тесты
- интеграционный тест
- тест на большом файле (100000 строк)

Можно запускать отдельно

## Docker

Сборка образа
```bash
docker build -t logger .
```
Запуск
```bash
docker run --rm \
  -v "$PWD/example_data/data.txt:/data.txt:ro" \
  logger /data.txt
```
